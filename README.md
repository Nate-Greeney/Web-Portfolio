Running locally (Vite + React)

- Prereqs: Node.js 18+ and npm
- Install deps:

```bash
npm install
```

- Start dev server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```

- Preview production build:

```bash
npm run preview
```

Honeybadger

- Honeybadger is configured in [src/main.jsx](src/main.jsx) via `@honeybadger-io/react`.
- To test error capturing, click the "Trigger React test error" button in the app.
